# Steps
- In the feature file select the values that needs to be parameterized
- Put the values within conical brackets or double quotes(Parameterization)
- Change Scenario to Scenario Outline
- Add Examples section
- Add data for the parameters in the Examples section
- Update step definition to get values from feature file Examples section
- Run and verify
